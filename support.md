## Hosting and support

Some MOLGENIS servers can be used free, and private
MOLGENIS servers can be self-hosted or provided fee for service by the
**Genomics Coordination Center (GCC)**, University Medical Center Groningen. Also support is available on open source fair use or project funded basis. Please contact us for details.

email: <a href="mailto:support@molgenis.org">support@molgenis.org</a>
<br/>conditions: [Acceptable Use Conditions(AUP)](/attachments/MOLGENIS_AUP.pdf), [Example service agreement](/attachments/MOLGENIS_DVO_annex4_20201120.pdf)

## Documentation

Find documentation for each MOLGENIS tool:
- [Platform (EMX2)](https://molgenis.github.io/molgenis-emx2/#/molgenis/) — data platform manual
- [Catalogue](https://molgenis.github.io/molgenis-emx2/#/catalogue/) — catalogue user guide
- [Variant Interpretation Pipeline (VIP)](https://molgenis.github.io/vip/) — installation & usage
- [Armadillo (DataSHIELD)](https://molgenis.github.io/molgenis-service-armadillo/) — federated analysis manual
- [HPC clusters](https://docs.gcc.rug.nl/) — cluster documentation

The full list of applications can be found at the MOLGENIS [GitHub](https://github.com/molgenis).

## Free services

MOLGENIS provides public services with our partners that are free to use:
- [Data catalogue](https://molgeniscatalogue.org/) in collaboration with many [consortia](/partners.html)
- [Biobank catalogue](http://directory.bbmri-eric.eu/) via BBMRI-ERIC

## Controlled access service
MOLGENIS partners also provide controlled access services:
- HPC compute cloud at [UMCG](https://umcgresearch.org/w/high-performance-computing)
- DataSHIELD armadillo network
